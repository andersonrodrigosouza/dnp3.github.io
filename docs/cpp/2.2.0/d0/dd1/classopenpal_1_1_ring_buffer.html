Detailed Description

template<uint8_t N>
class openpal::RingBuffer< N >

A byte-oriented ring buffer.

Interrupt-safe (w/o disabling interrupts) for a single producer and single consumer, one reading and one writing.

N must be a power of 2, and is enforced via static assert.
